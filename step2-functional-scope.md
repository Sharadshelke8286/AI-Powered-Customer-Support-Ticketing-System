# Step 2: Functional Scope

## Objective

The functional scope defines the main features and capabilities of the AI-Powered Customer Support Ticketing System implemented using Salesforce.

## Overview

The system is designed to help organizations efficiently manage customer support cases by automating ticket classification, priority assignment, SLA tracking, and agent assistance.

The platform provides a centralized environment where support agents, managers, and administrators can manage support operations effectively.

## Core Functional Features

### 1. Case Creation

Users can create customer support cases by entering issue details such as customer information, issue description, issue type, and priority level.

### 2. Automatic Case Classification

The system automatically categorizes cases based on issue type to ensure proper handling and faster resolution.

### 3. Priority Assignment

Each support case is assigned a priority level such as:

* High
* Medium
* Low

This helps support teams handle urgent issues first.

### 4. SLA Tracking

The system calculates Service Level Agreement (SLA) due dates for each case to ensure timely resolution.

### 5. Agent Assistance

Support agents receive suggested actions that help them resolve cases efficiently.

### 6. Case Status Management

Support agents can update the status of cases such as:

* New
* In Progress
* Pending
* Resolved
* Closed

### 7. Customer Communication

Agents can communicate with customers while resolving issues and provide updates regarding case progress.

### 8. Agent Feedback

Support agents can provide feedback on automated suggestions to evaluate the system's effectiveness.

## Role-Based Access Control

The system supports different user roles:

### Support Agents

* Manage and update support cases
* View suggested actions
* Communicate with customers

### Support Managers

* Monitor case performance
* View reports and dashboards
* Analyze support team productivity

### System Administrators

* Configure system automation
* Manage security settings
* Maintain data models and workflows

## Expected Outcome

The system improves customer support efficiency by automating repetitive tasks, improving ticket management, and providing visibility into support performance.
